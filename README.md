# Dashboard Mahasiswa PNB

UTS Integrasi Sistem - Politeknik Negeri Bali

## 🎯 Fitur

1. Total Mahasiswa PNB (Genap 2025/2026)
2. Mahasiswa TI TRPL Kelas 6C (25 mahasiswa)
3. Jumlah Mahasiswa per Jurusan
4. Jumlah Mahasiswa per Program Studi
5. Total Program Studi di PNB

## 🛠️ Tech Stack

- React 19
- Vite
- TailwindCSS
- PNB SION API

## 🚀 Cara Menjalankan

```bash
npm install
npm run dev
```
